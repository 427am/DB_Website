Setup Instructions
1. Download the Project Files

After downloading the project ZIP, extract all files.
Inside the extracted folder you should see a folder named realestate_site.

2. Install XAMPP

Download XAMPP from:
https://www.apachefriends.org/download.html

Install it on your computer.

3. Move the Project Folder

After XAMPP is installed, open the XAMPP installation directory.
Inside that directory, open the folder named htdocs.

Move the realestate_site folder into the htdocs folder.
This is where XAMPP stores all website files.

4. Start Apache and MySQL

Open the XAMPP Control Panel.
Start Apache and MySQL.

5. View the Website

Open your browser and go to:

http://localhost/realestate_site


The site should now load.
